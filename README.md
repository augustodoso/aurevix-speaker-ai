# Aurevix Speaker AI 🚀

Aurevix Speaker AI is an AI-powered presentation platform built with React, FastAPI and OpenAI.

The project allows users to:

- generate presentations with AI
- export presentations to PowerPoint (.pptx)
- edit AI-generated slides
- save presentations by user
- upload PDFs/slides
- receive audience questions in real time
- use AI Copilot with RAG/contextual responses
- interact through QR Code audience access

Tech Stack:
- React
- FastAPI
- OpenAI API
- WebSockets
- SQLAlchemy
- JWT Authentication
- Render
- Vercel

This project was built as a real SaaS-style AI application focused on smart presentations and audience interaction.
